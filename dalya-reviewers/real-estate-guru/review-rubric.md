# Real Estate Guru Review Rubric

## 1. Sales usefulness — score 1-10

Look for:

- Does this help the agent book viewings, win listings, follow up, or close?
- Does it reduce real admin work?
- Does it prioritize money-making activity?
- Does it surface urgency?

## 2. Dubai resale correctness — score 1-10

Look for:

- Ready property vs off-plan resale differences.
- Fees framed correctly.
- No unsupported claims about availability, ROI, negotiability, mortgage, transfer, or legal requirements.
- Correct use of transaction concepts.

## 3. Buyer qualification — score 1-10

Look for:

- Right questions.
- Right order.
- Not too many questions.
- Hot leads detected and handed off.
- Low-intent leads routed sensibly.

## 4. Agent workflow fit — score 1-10

Look for:

- WhatsApp reality acknowledged.
- CRM/admin friction reduced.
- Dashboard tells agent what to do next.
- Follow-up suggestions are useful.
- Viewings and owner follow-ups are actionable.

## 5. Dashboard daily value — score 1-10

Look for:

- Clear morning workflow.
- Hot leads and pending actions visible.
- Listing/document gaps visible.
- Owner outreach support.
- No dead “analytics for the sake of analytics.”

## Required output additions

Include:

- Would I use this daily? Why/why not?
- Would this help close deals? How specifically?
- Missing fields for a real agent.
- Buyer-facing claims that require verification.
- Features that sound good but agents would ignore.
- Top 5 changes to make Dalya more useful tomorrow morning.
