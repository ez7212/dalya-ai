# Review All Command

Use before a major demo, launch checkpoint, or MVP scope review.

## Read first

- `dalya-reviewers/shared-context.md`
- `dalya-reviewers/review-session-rules.md`
- `dalya-reviewers/file-routing.md`
- `dalya-reviewers/templates/review-output-template.md`
- All four reviewer persona/rubric files.
- Relevant knowledge packs and checklists.

## Scope

Review the current MVP from all four perspectives:

1. Chatbot Master.
2. Real Estate Guru.
3. UX Designer.
4. Security Researcher.

## Instructions

Do not change code.

Return:

1. Separate findings for each reviewer.
2. Combined Red/Yellow/Green launch confidence.
3. Top 10 cross-functional launch risks.
4. Features to cut or delay.
5. Must-have additions before launch.
6. Linear-ready tasks.
