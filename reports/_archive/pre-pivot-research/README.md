## What survives substantively

The **market sizing data** in the demand-supply files (UAE property search volumes, segmentation by buyer profile, language distribution) is still factually accurate. Useful if anyone needs to size markets, validate buyer demographics, or build pitch decks for the B2B sale.

The **gap-analysis logic** of "where do incumbents fail" still applies — but the gap Dalya now closes is *agent workflow* (not buyer information). The gaps surfaced in `gap-analysis.md` are still real; they're just no longer the gaps we're filling.

The **per-language demand research** is most useful as input for the bot's multilingual prompt scope. We supported EN/AR/RU/HI in v1; Mandarin is in Pillar 1 backlog. The research here gives us a baseline for prioritizing Mandarin investment if it lands later.

Otherwise: historical reference only.
